**Goal:** Make the `suggest` model a study-level, reproducible setting instead of only a CLI flag, and fail with a clear instruction (not a raw HTTP 401) when an LLM model is requested without `OPENROUTER_API_KEY`.

**Architecture:** `Study` gains a `suggest_model` field (from `study.yaml`, default `z-ai/glm-5.2`). The `suggest` CLI's `--model` becomes optional and *overrides* the study setting when given (precedence: flag > study.yaml > built-in default). `openrouter_complete` raises a friendly `RuntimeError` if no key is available, caught by the command for a clean exit. `--model rules` stays the offline baseline.

**Tech Stack:** Python ≥3.11, pydantic, typer. No new dependencies. Tests fully offline.

**Environment:** uv. Tests via `uv run pytest -q`, lint via `uv run ruff check vogue tests`.

Work on branch `v0.2-suggest-config`. Commit after each task. Additive except the `--model` default moves from the CLI into `Study` (so `--model` defaults to `None`).

Reference spec: §3 (reproducibility — record which model coded a study).

Change the `--model` option default to `None` and resolve it from the study; construct the suggester *before* fetching, inside a try/except. Replace the `suggest` command with:

```python
@app.command()
def suggest(study_dir: str, source: str = typer.Option(...), term: str = typer.Option(...),
            model: str = typer.Option(None,
                                      help="OpenRouter model slug (default: study.yaml suggest_model), "
                                           "or 'rules' for the offline baseline"),
            limit: int = typer.Option(None, help="cap records fetched")):
    """Pre-label uncoded records with an LLM (via OpenRouter) into a side file.

    Suggestions are never authoritative — `code` shows them as defaults for human confirmation.
    The model comes from study.yaml (`suggest_model`) unless --model overrides it.
    Needs OPENROUTER_API_KEY for real models; use --model rules for an offline no-op baseline.
    """
    study = Study.load(study_dir)
    resolved_model = model or study.suggest_model
    try:
        suggester = _make_suggester(resolved_model)
    except RuntimeError as e:
        typer.echo(str(e))
        raise typer.Exit(1)
    records = pipeline.fetch_term(study, source, term, limit=limit)
    kept = pipeline.funnel_for_records(records, study.keep_fields).kept
    coded = CodingStore(study.codings_dir / f"{term}.csv").coded_keys()
    sstore = SuggestionStore(study.codings_dir / f"{term}.suggestions.csv")
    have = sstore.suggested_keys()
    todo = [r for r in kept if r.key not in coded and r.key not in have]
    today = datetime.date.today().isoformat()
    for r in todo:
        sstore.append(Suggestion(key=r.key, term=term, suggested=suggester.suggest(r, term),
                                 model=resolved_model, suggested_at=today))
    typer.echo(f"suggested {len(todo)} records ({len(have)} already had suggestions)")
```
